# PupaMVCF

### Description

- This simplify web-framework is built on top of .NET 8 and C# 12, and it is designed to provide a simple and efficient
  way to build web applications using the Model-View-Controller (MVC) architectural pattern. It offers a range of
  features and tools to help developers create robust and scalable web applications with ease.

### OS

- Cross-platform is supported for now.

### Features

- Cross-platform
- MVC
- gRPC
- Middleware (Developing)
- Views
- Dependency Injection
- Logging
- Configuration
- Models
- Controllers
- Routing
- Validations (Developing)
- Error handling
- Xss
- SqlInjection
- DaemonProcessing
- MicroServices
- Cookies
- Sessions (Kestrel)
- Kestrel
